This project follows the [all-contributors](https://github.com/all-contributors/all-contributors) specification. Contributions of any kind welcome!

## Recognition Levels

### 🥇 Gold Contributors
- Multiple major features or significant refactoring
- Recognized in README and releases
- Eligible for core team

### 🥈 Silver Contributors
- Bug fixes and smaller features
- Active in discussions and reviews
- Listed in CONTRIBUTORS.md

### 🥉 Bronze Contributors
- Documentation improvements
- Test additions
- Issue reporting
- Listed in releases

## Recognition Process

1. Make a contribution (code, docs, tests, etc.)
2. Your PR is merged
3. Request recognition in the merged PR or via issue
4. We'll add you using [@all-contributors](https://allcontributors.org/)

## Contribution Types

### 💻 Code
Bug fixes, features, optimizations, refactoring

### 📖 Documentation
README updates, API docs, tutorials, deployment guides

### ⚠️ Tests
Unit tests, integration tests, test improvements

### 🐛 Bug Reports
Well-documented issue reports

### 🤔 Ideas
Feature suggestions, architectural improvements

### 🎨 Design
UI/UX improvements, visual enhancements

### 🔍 Review
Code reviews, constructive feedback

### 📢 Content
Blog posts, tutorials, conference talks

Open an [issue](https://github.com/yourusername/nutrition-recommender/issues) or start a [discussion](https://github.com/yourusername/nutrition-recommender/discussions).
